# Personalized AI Study Planner Agent

## Overview
This project aims to build an AI-powered personalized study planner agent that dynamically generates and adapts study schedules based on student performance, syllabus difficulty, and deadlines.

## Features
- Personalized study schedule generation
- Adaptive rescheduling based on performance
- AI agent-based decision making
- Reinforcement learning for time optimization
- NLP-based user interaction

## Tech Stack
- Python
- Scikit-learn
- Reinforcement Learning (Q-learning)
- NLP (Sentence Embeddings)
- Streamlit (UI)
